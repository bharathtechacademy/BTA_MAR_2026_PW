// CI/CD => Continuous integration and continuous deployment. 

//Continuous integration => Integrating with cloud repository to capture the latest changes every time 
//Continuous deployment => Execute the latest changes and produce the results. 


//Different stages of CI/CD. 

//There are mainly five different stages in the CI/CD pipeline. 

//Stage 1 : Source Code Management 

Copy the latest changes or check out the latest changes from the cloud repository. And get latest pull request and merge request details. 

//Stage 2 : Build Stage 

Compile the latest changes extracted from the source code management tool and resolve the dependencies and validate the build. 

//Stage 3 : Test Stage 

Perform the unit testing on the latest changes copied from the source code management tool and understanding the code quality. 

//Stage 4 : Deployment Stage 
Deploy the code from one environment to the other environment. 

//Stage 5 : Monitoring and feedback stage 
Monitor each and every change and log the results and send feedback to the user.
