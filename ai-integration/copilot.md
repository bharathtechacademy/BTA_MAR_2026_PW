# What is Copilot? 
Copilot is an AI-powered coding agent available as an extension within the IDE. 

# How GitHub Copilot will work and what are all the different features of GitHub Copilot? 
Github Copilot is powered by LLMs, and it is going to understand:
- your current file or your current project structure
- your existing framework pattern
- the comments written by the user
- the method names
- all the libraries imported within the project
and suggest the code.

# Different features of Copilot ?

1. Inline Code Suggestions : Whenever we are going to write the code, it is going to fill the remaining part by itself. 

2. Co-Pilot Chat (Ask mode): Co-pilot chat ask modi is all about getting the suggestions. 

3. Co-Pilot Chat (Agent mode): Co-pilot agent mode is all about completing the task by itself. Instead of suggestion, it is going to directly write the program. Update your program file. 

4. Co-Pilot Chat (Plan mode): Co-pilot plan mode is all about creating an execution strategy before making the changes. Meaning, Co-pilot is going to analyze the requirement, break it into multiple steps, and then it is going to give you the strategy to complete. 

5. Test Generation  : It can actually generate all the possible test cases, assertions, and reusable methods during the automation development process. 

6. Generate the documentation. : Copilot can actually help us to prepare detailed documentation comments for each and every common method that we are creating. 

7. Can be integrated with MCP : Scopilot has the capability to integrate MCP to complete the task and also to handle the tools together.
